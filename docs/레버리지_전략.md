## 🛡️ 리스크 관리

### 1. 레버리지 리스크

| 리스크 | 대응 방안 |
|--------|----------|
| **금리 상승** (5% → 7%) | 변동 → 고정금리 전환<br>퀀트 자본 축소로 대출 상환 |
| **주식 폭락** (-30%) | 손절매: -20% 시 포지션 50% 청산<br>LTV 70% 이하 유지 |
| **퀀트 실패** | 퀀트 비중 축소 37.5% → 20%<br>S&P500 ETF로 전환 |
| **유동성 위기** | 주식 2억 8,000만원 급매 가능<br>신용대출 여력 4,300만원 |

### 2. Circuit Breaker (강제 중단)

```python
circuit_breaker_rules = {
    '일일 손실': '순자산의 -3% (2,700만원)',
    '주간 손실': '순자산의 -5% (4,500만원)',
    '월간 손실': '순자산의 -10% (9,000만원)',
    '분기 손실': '순자산의 -15% (1억 3,500만원)',
    '연속 손실': '5거래일 연속 손실 시 1주 중단',
    '드로우다운': '최고점 대비 -20% 시 전략 재검토'
}

# 트리거 시 행동
if circuit_breaker_triggered:
    1. 모든 포지션 즉시 청산
    2. 1-2주 휴식
    3. 전략 전면 재검토
    4. 백테스트 재실행
    5. 파라미터 재조정 후 소액 재시작
```

### 3. 반드시 지켜야 할 3가지

1. **비상금 확보**: 대출 이자 6개월치 (240만원)
2. **손절매 철칙**: -20% 도달 시 무조건 포지션 50% 청산
3. **단계적 실행**: 1년 검증 후 레버리지 (조급함 금물)
